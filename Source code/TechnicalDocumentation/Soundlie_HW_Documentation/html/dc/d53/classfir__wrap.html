ieee
Library

This is the wrapper for the first filter stage of the FIR decimation filter design.

This code block is a wrapper which connects the first stage FIR filter (fir1) to the top-entity of the decimation filter design (top_fir).

Author
Malin Eliasson
Version
2.0
W INTEGER := 16
Generic

Sets the width (number of bits) used for numbers in the wrapper.

This wrapper entity takes inputs from the top-entity of the decimation filter design (top_fir). It also outputs signals to top_fir. The outputs of the wrapper are generated by the FIR filter sub-block (fir1) connected to the wrapper.
