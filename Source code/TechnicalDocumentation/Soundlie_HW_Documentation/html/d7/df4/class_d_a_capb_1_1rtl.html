Detailed Description

This the APB interface architecture.

This is a modified version of the Gaisler APH tutorial code. It interfaces with the APB in the way that is done in the tutorial, with slight modification to the registers(just one register). The interface has three sub-modules added to the original design.

  1. PWM interface for converting samples and outputting straight to the on-board reconstruction filter
  2. SPI communication message generation(DAC_interface).
  3. clock enable that generates the serial communication clock for the SPI communications.
