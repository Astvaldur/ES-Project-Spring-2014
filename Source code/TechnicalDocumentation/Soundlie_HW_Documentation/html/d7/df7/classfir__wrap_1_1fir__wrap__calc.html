Detailed Description

This is the architecture of the first wrapper of the first decimation stage of the decimation filter design.

This architecture of the first wrapper of the first decimation stage contains all the code which describes the functionality of the wrapper. It contains a FIR filter sub-block, fir1, which does the filter calculations. The wrapper takes the output from the filter and sends it to the top-entity.
